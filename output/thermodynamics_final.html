고등학생 수준으로 확장하면

• 총합은 양수!

완전 결정의 엔트로피는 절대영도에서 0에 접근해.
lim(T→0) S = 0 (완전 결정의 경우)

왜 중요해?

이게 있어야 엔트로피에 절대적인 기준점이 생겨. 2단계에서 배운 ΔS는 "변화량"이었지? 3법칙 덕분에 S 자체를 말할 수 있게 돼.

대학/전문가 수준

정의 (Definition)

절대영도(Absolute Zero): 열역학적 온도 척도의 하한으로, 0 K (-273.15°C). 계에서 추출 가능한 열에너지가 없는 상태. 단, 양자역학적 영점에너지(zero-point energy)는 존재해.

출처: Callen, H.B. (1985). Thermodynamics and an Introduction to Thermostatistics, 2nd ed., Wiley. Chapter 11.

양자역학적 함의:
하이젠베르크 불확정성 원리 때문에 입자는 절대영도에서도 완전히 멈출 수 없어. 이게 영점에너지(zero-point energy)야.

2.1.1 르장드르 변환의 직관

대학 수준 - 르장드르 변환

정의 (Definition)

르장드르 변환(Legendre Transformation): 함수의 독립변수를 그 변수에 대한 편미분 값(켤레 변수)으로 바꾸는 수학적 기법.

U(S,V)에서 S를 T = (∂U/∂S)_V로 바꾸려면:

F = U - TS

이게 헬름홀츠 자유에너지(Helmholtz Free Energy)야!

2.1.2 네 포텐셜 총정리

포텐셜 기호 정의 자연변수 미분형
내부에너지 U 기본 S, V dU = TdS - PdV
헬름홀츠 F U - TS T, V dF = -SdT - PdV
엔탈피 H U + PV S, P dH = TdS + VdP
깁스 G H - TS T, P dG = -SdT + VdP

암기 팁

      S
      |
  U --- H
  |     |
  F --- G
      |
      T

가로: V ↔ P (압력-부피) | 세로: S ↔ T (엔트로피-온도)

2.1.3 자유에너지의 물리적 의미

헬름홀츠 자유에너지 F = U - TS:

정의:
등온(T 일정) 과정에서 계가 할 수 있는 최대 일.

분해해보자:

• U: 총 내부에너지

• TS: "잠겨있는" 에너지 (엔트로피 때문에 질서있는 일로 못 바꿈)

• F = U - TS: 진짜 쓸 수 있는 에너지 = "자유" 에너지

깁스 자유에너지 G = H - TS:

정의:
등온등압(T, P 일정) 과정에서 계가 할 수 있는 최대 비팽창 일 (non-expansion work).

왜 화학자들이 좋아하냐면:

• 실험실: 대기압(P 일정) + 항온조(T 일정)

• ΔG < 0이면 자발적 반응!

• ΔG = 0이면 평형!

출처: Atkins, P. & de Paula, J. (2014). Atkins' Physical Chemistry, 10th ed., Oxford. p. 128-135.

전문가 수준 - 자발성 판정

조건 자발성 판정 기준
고립계 dS ≥ 0 (엔트로피 증가)
등온등적 (T, V 일정) dF ≤ 0 (헬름홀츠 감소)
등온등압 (T, P 일정) dG ≤ 0 (깁스 감소)

유도 (스스로 해보기!)

등온등압에서 dS_total = dS_system + dS_surr ≥ 0

주위가 받은 열: q_surr = -q_sys (등압에서 q_sys = dH)
dS_surr = q_surr/T = -dH/T

따라서:
dS_sys - dH/T ≥ 0
TdS_sys - dH ≥ 0
-(dH - TdS_sys) ≥ 0
-dG ≥ 0 → dG ≤ 0

유도 과정을 따라가보면 개념이 훨씬 선명해져 | Unsplash

2.2 맥스웰 관계식 (Maxwell Relations)

2.2.1 수학적 기초: 완전미분

함수 f(x, y)의 전미분:

df = (∂f/∂x)_y dx + (∂f/∂y)_x dy

간단히 df = M dx + N dy라고 쓰면,

완전미분 조건:

(∂M/∂y)_x = (∂N/∂x)_y

생각해볼 질문 4
왜 이게 성립할까? 힌트: ∂²f/∂x∂y와 ∂²f/∂y∂x의 관계는?

2.2.2 네 개의 맥스웰 관계식 유도

dU = TdS - PdV에서:

• M = T, N = -P

• x = S, y = V

완전미분 조건 적용:

(∂T/∂V)_S = -(∂P/∂S)_V

나머지 세 개도 같은 방식으로:

포텐셜 미분형 맥스웰 관계식
U dU = TdS - PdV (∂T/∂V)_S = -(∂P/∂S)_V
F dF = -SdT - PdV (∂S/∂V)_T = (∂P/∂T)_V
H dH = TdS + VdP (∂T/∂P)_S = (∂V/∂S)_P
G dG = -SdT + VdP -(∂S/∂P)_T = (∂V/∂T)_P

노란색으로 표시된 두 개가 가장 자주 쓰여! (측정 가능한 양들만 포함)

전문가 수준 - 활용 예시

문제: 이상기체의 (∂S/∂V)_T를 구하라.

풀이:
맥스웰: (∂S/∂V)_T = (∂P/∂T)_V

이상기체 상태방정식 PV = nRT에서:
P = nRT/V
(∂P/∂T)_V = nR/V

따라서: (∂S/∂V)_T = nR/V

이걸 적분하면 2단계에서 배운 ΔS = nR ln(V₂/V₁) (등온팽창)이 나와!

출처: Zemansky, M.W. & Dittman, R.H. (1997). Heat and Thermodynamics, 7th ed., McGraw-Hill. Chapter 10.

2.3 상전이와 클라우지우스-클라페이론 방정식

2.3.1 상(Phase)이란?

기초부터 시작하자:
물의 세 가지 모습: 얼음 ↔ 물 ↔ 수증기

같은 H₂O인데 왜 다르게 보여? 분자 배열이 다르니까!

정의 (Definition)

상(Phase): 물리적, 화학적으로 균일하고, 명확한 경계로 다른 부분과 구분되는 물질의 상태.

상전이(Phase Transition): 한 상에서 다른 상으로 변하는 것.

전이 이름 예시
고 → 액 융해(melting) 얼음 → 물
액 → 기 기화(vaporization) 물 → 수증기
고 → 기 승화(sublimation) 드라이아이스

같은 H₂O, 다른 배열 | Unsplash

2.3.2 상평형도 (Phase Diagram)

상평형도: 삼중점(Triple Point)과 임계점(Critical Point) | Wikimedia Commons

삼중점(Triple Point): 세 상이 동시에 평형을 이루는 유일한 (T, P) 점

임계점(Critical Point): 액체와 기체의 구분이 사라지는 점. 이 이상의 T, P에서는 초임계 유체

물의 데이터:

• 삼중점: T = 273.16 K, P = 611.73 Pa

• 임계점: T = 647.1 K, P = 22.064 MPa

2.3.3 상평형 조건

생각해볼 질문 5
0°C, 1기압에서 얼음과 물이 공존해. 이때 두 상 사이에서 뭐가 같아야 평형이야?

답: 깁스 자유에너지가 같아야 해!

G_solid = G_liquid

유도:
등온등압에서 평형이면 dG = 0.
만약 G_s < G_l이면 → 모두 고체가 됨 (안정)
만약 G_s > G_l이면 → 모두 액체가 됨 (안정)
G_s = G_l일 때만 → 두 상이 공존!

2.3.4 클라우지우스-클라페이론 방정식

유도 (함께 따라가보자)

두 상 α, β가 평형일 때: G^α(T, P) = G^β(T, P)

(T, P)가 살짝 변해도 여전히 평형이면:
dG^α = dG^β
-S^α dT + V^α dP = -S^β dT + V^β dP

정리하면:
dP/dT = (S^β - S^α)/(V^β - V^α) = ΔS/ΔV

상전이에서 ΔS = ΔH/T (등온 과정)이므로:

dP/dT = ΔH / (T ΔV)

클라우지우스-클라페이론 방정식 (Clausius-Clapeyron Equation)

출처: Kondepudi, D. & Prigogine, I. (2014). Modern Thermodynamics, 2nd ed., Wiley. Chapter 6.

생각해볼 질문 6
대부분의 물질은 압력을 높이면 녹는점이 올라가. 그런데 물은 압력을 높이면 녹는점이 내려가. 왜 그럴까?

힌트: 클라우지우스-클라페이론에서 dP/dT의 부호를 생각해봐.

• 융해: ΔH > 0 (항상 양수, 열 흡수)

• 대부분 물질: V_liquid > V_solid → ΔV > 0 → dP/dT > 0

• 물: V_liquid < V_solid (얼음이 물에 뜬다!) → ΔV < 0 → dP/dT < 0

이게 왜 중요해?

• 스케이트: 압력으로 얼음이 녹아 윤활 작용? (실제론 마찰열이 주 원인)

• 빙하: 바닥의 높은 압력에서 녹아 흐름

2.4 초임계 유체와 산업 응용

정의

초임계 유체(Supercritical Fluid, SCF): 임계점 이상의 온도와 압력에서 존재하는 물질. 액체와 기체의 경계가 없어 독특한 물성을 가져.

초임계 CO₂ (sc-CO₂):

• 임계점: Tc = 31.1°C, Pc = 7.38 MPa

• 장점:

- 낮은 임계온도 (열에 약한 물질 추출 가능)

- 무독성, 비가연성, 잔류물 없음

- 압력 조절로 용해도 조절 가능
